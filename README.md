## Usage

To launch the project:

### Copy the repo

```bash
git clone https://github.com/dmytro-nesteruk/auto-settle-coding-challenge.git
```

### Install dependencies

```bash
pnpm install
```

### Run the project

```bash
pnpm dev
```
